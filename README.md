AGRID: Analytical Greenery Remote Integrated Device

----------- Project Overview -----------
AGRID (Analytical Greenery Remote Integrated Device) is an integrated system designed to modernize agricultural practices through Internet of Things (IoT) technology. 
The system utilizes an ESP32 microcontroller to monitor soil moisture levels in nursery plants and transmits this data to a dedicated mobile application. 
This solution provides users with real-time environmental insights and the capability to automate irrigation, ensuring optimal plant health and resource efficiency.
